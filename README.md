# CPP Module 03

## 📚 Description

This project introduces the concept of **inheritance in C++**, focusing on how classes can derive from other classes to reuse and extend functionality.
It focuses on how to structure class hierarchies, manage constructor/destructor chains, and apply polymorphic-like behavior through inheritance (without full runtime polymorphism yet).
The module is entirely based on the **C++98 standard**.

## 🧠 Key Concepts

- Class inheritance (base / derived classes)
- Access specifiers (`public`, `protected`, `private`)
- Constructor and destructor chaining
- Reusing base class functionality
- Method overriding
- Code reuse through inheritance
- Proper object initialization in inheritance trees
